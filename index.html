Quelques raccourcis clavier LINUX

1.Général

  1. Win : Ouvre le menu Application
  2. Win+D : Montrer Bureau
  3. Win+E : Ouvre le gestionnaire de fichier
  4. Alt+F2 : Ouvre la boîte de dialogue d'éxécution
  5. Ctrl+Alt+T : Ouvre le terminal
  6. Ctrl+C : Copier
  7. Ctrl+V : Coller

2.Navigation dans les répertoires

  1. Ctrl+Shift+N : Nouveau fichier
  2. Ctrl+L : concentrer la barre d'adresse
  3. Alt+Home : Accéder au répertoire Home
  4. Alt+Arrow Up : Naviguer jusqu'à un répertoire
  5. Alt+Arrow Right/Arrow Left : Naviguer un répertoire avant/arrière
